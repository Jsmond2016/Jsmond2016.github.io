# javaScript-study

> 详细内容参考： [现代 JavaScript 教程](https://zh.javascript.info/)

JavaScript 内容主要为几个部分

- es5
- es6
- typeScript

其中，es5 的重点内容为：

- 原型和原型链
- 闭包
- 作用域（执行上下文）
- this
- new 关键字

其中，es6 的重点内容为：

- 普通使用，参考 [ECMAScript 6 入门](https://es6.ruanyifeng.com/)
- 类
- Promise, async, await
- Generator
- 箭头函数等
